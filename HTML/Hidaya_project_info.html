Hidaya (Work In Progress)

Hidaya is a personal project aimed at creating a modern, user-friendly Islamic platform that provides access to the Quran, Hadith collections, Duas, and Tafsir. The app is designed with personalized features like a prayer tracker, Ramadan tools, and user reflections. It includes AI-powered search party and suggestions to help users explore Islamic content with ease and relevance. The goal is to support spiritual growth with thoughtful design, accessible language, and a calm visual aesthetic that blends traditional values with modern web practices.
