Midnight Noir

Midnight Noir is a two-page coffee shop website built with HTML, CSS, and JavaScript as part of a university coursework project. It includes a homepage and a menu page, structured with semantic HTML and styled for clarity and usability. The project focused on building a responsive layout and ensuring smooth user interaction using core web technologies without external frameworks. Through this work, I gained hands-on experience in frontend structure, page layout, and basic interactivity.
